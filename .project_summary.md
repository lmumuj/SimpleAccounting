## 📊 数据结构

### 账目记录
```javascript
{
  id: Number,              // 唯一ID
  type: String,            // 'expense' | 'income'
  amount: Number,          // 金额
  categoryId: Number,      // 类别ID
  remark: String,          // 备注
  date: String,            // 日期 YYYY-MM-DD
  createTime: String       // 创建时间
}
```

- 支出类别：8种（餐饮、交通、购物、娱乐、医疗、教育、居住、其他）
- 收入类别：5种（工资、奖金、投资、兼职、其他）

## 🚀 快速开始

### 前置条件
1. 安装微信开发者工具
2. 准备6个TabBar图标（81x81 PNG）

### 安装步骤
1. 在 `images/` 目录放入图标文件
2. 打开微信开发者工具
3. 导入项目目录
4. 点击编译运行

## ⚠️ 注意事项

### 重要提醒
1. **图标资源必须准备**：项目需要6个底部导航图标才能正常运行
2. **数据存储在本地**：更换设备数据不会同步，建议定期导出备份
3. **使用微信开发者工具**：必须在微信开发者工具中运行，不能直接打开

### 已知限制
1. 不支持跨设备数据同步
2. 不支持多人共享账本
3. 暂不支持搜索功能
4. 暂不支持预算管理
