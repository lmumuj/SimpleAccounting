# 简单记账 - Python桌面预览版

## 📖 说明

这是一个使用Python和Tkinter实现的桌面版记账应用，用于预览Android应用的功能和界面。

## ✨ 功能特性

- ✅ 完整的记账功能（添加、编辑、删除）
- ✅ 数据统计（周、月、年）
- ✅ 分类展示（支出和收入）
- ✅ 数据持久化（JSON格式）
- ✅ 数据导出（CSV格式）
- ✅ 友好的界面设计
- ✅ 支持筛选（全部/支出/收入）

## 🚀 快速开始

### 环境要求

- Python 3.6 或更高版本
- Tkinter（通常Python自带）

### 安装步骤

#### 1. 检查Python版本

```bash
python --version
# 或
python3 --version
```

确保版本是 3.6 或更高。

#### 2. 运行应用

**Windows用户：**

```bash
python preview_app.py
```

**macOS/Linux用户：**

```bash
python3 preview_app.py
```

或者直接双击 `preview_app.py` 文件（如果系统配置了Python关联）。

### 使用方法

#### 添加记账

1. 点击右下角的"+ 记账"按钮
2. 选择类型（支出/收入）
3. 输入金额
4. 选择类别
5. 可选填写备注
6. 选择日期
7. 点击"保存"

#### 查看记账

- 主界面显示所有账目列表
- 可以筛选显示全部/支出/收入
- 点击任意账目可以查看详情或编辑

#### 统计功能

1. 点击底部的"📊 统计"按钮
2. 选择时间周期（本周/本月/本年）
3. 查看收支总览和分类统计

#### 数据管理

1. 点击底部的"👤 我的"按钮
2. 查看统计信息
3. 导出数据（复制到剪贴板）
4. 清空数据（需确认）

## ⚙️ 自定义配置

### 修改类别

编辑 `preview_app.py` 文件中的 `Category` 类：

```python
class Category:
    EXPENSE_CATEGORIES = [
        {'id': 1, 'name': '餐饮', 'icon': '🍔'},
        # 添加或修改支出类别
    ]

    INCOME_CATEGORIES = [
        {'id': 101, 'name': '工资', 'icon': '💰'},
        # 添加或修改收入类别
    ]
```

### 修改配色

编辑 `AccountingApp.__init__()` 方法中的 `self.colors` 字典：

```python
self.colors = {
    'primary': '#4A90E2',    # 主题色
    'expense': '#FF5252',    # 支出色
    'income': '#52C41A',     # 收入色
    'bg': '#F5F5F5',         # 背景色
    'card_bg': '#FFFFFF',    # 卡片背景
    'text': '#333333',       # 文字色
    'text_secondary': '#999999'  # 辅助文字色
}
```

## 🔧 故障排除

### 问题：运行提示"Tkinter not found"

**解决方案：**
- Windows：重新安装Python时勾选"tcl/tk and IDLE"选项
- Linux：`sudo apt-get install python3-tk`
- macOS：通常已包含，如无则安装 `brew install python-tk`

### 问题：数据丢失

**解决方案：**
- 检查是否有 `transactions.json` 文件
- 文件可能被移动或删除
- 建议定期备份数据文件

### 问题：界面显示异常

**解决方案：**
- 确保Python版本是3.6+
- 尝试更新Python到最新版本
- 检查系统是否支持Tkinter

## 📝 注意事项

1. **数据安全**：数据保存在本地JSON文件中，请定期备份
2. **日期格式**：必须使用 YYYY-MM-DD 格式
3. **金额格式**：仅支持数字和小数点
4. **跨平台**：支持Windows、macOS、Linux
5. **离线使用**：完全不需要网络连接
