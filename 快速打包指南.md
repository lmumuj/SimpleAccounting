# 📱 简单记账应用 - 快速打包指南

## ⚡ 三种打包方法

### 方法一：使用 Android Studio（最推荐 ⭐⭐⭐⭐⭐）

这是最简单、最可靠的方法！

**步骤：**

1. **下载安装 Android Studio**
   - 访问：https://developer.android.com/studio
   - 下载并安装最新版本

2. **打开项目**
   - 启动 Android Studio
   - 选择 `File` -> `Open`
   - 选择文件夹：`c:\Users\Admins\Documents\记账`

3. **等待同步完成**
   - 打开项目后，Android Studio 会自动同步 Gradle
   - 等待右下角进度条完成（可能需要几分钟）

4. **构建 APK**
   - 点击菜单：`Build` -> `Build Bundle(s) / APK(s)` -> `Build APK(s)`
   - 等待构建完成

5. **获取 APK**
   - 构建完成后，点击通知栏中的 "locate"
   - 或者找到文件：`app\build\outputs\apk\debug\app-debug.apk`

**优点：**
- ✅ 自动处理所有依赖和配置
- ✅ 提供友好的错误提示
- ✅ 支持代码编辑和调试
- ✅ 可视化操作，无需命令行

### 方法二：使用打包脚本（需要先配置环境）

我已经为你准备了自动化脚本：`build_apk.bat`

**前提条件：**
- 必须先安装 JDK 17+
- 必须配置 Android SDK
- 网络连接正常（用于下载依赖）

**使用方法：**
```bash
# 双击运行
build_apk.bat
```

或者从命令行：
```bash
cd c:\Users\Admins\Documents\记账
build_apk.bat
```

脚本会自动：
- 检查 Java 版本
- 检查 Gradle 配置
- 执行构建命令
- 显示 APK 位置

### 方法三：手动命令行（高级用户）

如果你已经配置好了完整环境，可以手动执行：

## 🔧 如果命令行方式失败

```
解决：安装 Android Studio，它会自带正确的 JDK
```

**错误4：网络连接失败**
```
解决：检查网络连接，或配置 Gradle 镜像源
```

## 📦 APK 安装测试

### 方法1：通过 ADB 安装

```bash
# 1. 启用手机的开发者选项和 USB 调试
# 2. 用 USB 连接手机和电脑
# 3. 运行安装命令
adb install app\build\outputs\apk\debug\app-debug.apk

# 如果安装失败（可能已安装旧版本）
adb install -r app\build\outputs\apk\debug\app-debug.apk

# 卸载应用
adb uninstall com.simpleaccounting.app
```

### 方法2：直接传输文件

1. 将 `app-debug.apk` 传输到手机
2. 在手机上点击 APK 文件
3. 选择"安装"（可能需要允许"未知来源"）

## 🚀 发布到应用商店

### Debug 版本 → Release 版本

如果要发布到应用商店，需要：

1. **生成正式签名密钥**
   ```bash
   keytool -genkey -v -keystore my-key.keystore -alias my-alias -keyalg RSA -keysize 2048 -validity 10000
   ```

2. **配置签名**
   - 在 Android Studio 中：`Build` -> `Generate Signed Bundle / APK`
   - 选择刚才生成的密钥库

3. **构建 Release APK**
   - 使用上述工具生成签名后的 APK
