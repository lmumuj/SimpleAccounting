# 📱 小白专用教程 - 一步步教你打包APP

## 🎯 你现在需要做什么

简单说就是3个步骤：
1. 在网页上创建一个"仓库"（就像创建一个文件夹）
2. 把代码上传到这个仓库
3. 等待它自动生成APP安装包

**总共只需要15-20分钟！**

---

## 🌟 第一步：在 GitHub 上创建仓库（2分钟）

### 1.1 打开网页

在浏览器（比如谷歌浏览器、360浏览器）中输入这个网址：
```
https://github.com/new
```

或者：
1. 先打开 https://github.com
2. 点击右上角的 **+** 号
3. 选择 **New repository**

### 1.2 填写信息

你会看到一个页面，需要填写：

```
Repository name（仓库名称）: SimpleAccounting
Description（描述）: 简单记账应用
Public / Private（公开/私有）: 选择 Public（推荐）
```

**重要！重要！重要！**

下面这3个选项，**全部不要勾选**：
- ❌ Add a README file（不要勾选）
- ❌ Add a .gitignore（不要勾选）
- ❌ Choose a license（不要勾选）

### 1.3 点击创建

页面最下面有一个绿色的按钮，写着：
```
Create repository
```

点击它！

### 1.4 复制仓库地址

创建成功后，你会看到一个新的页面，页面上会显示类似这样的地址：

```
https://github.com/你的用户名/SimpleAccounting.git
```

**把这个地址复制下来，保存到记事本里！**

比如：
```
https://github.com/zhangsan/SimpleAccounting.git
```

---

## 💻 第二步：上传代码（5分钟）

### 2.1 打开命令窗口

**方法1：使用 Windows 搜索**
1. 按键盘上的 **Win 键**（田字格图标）
2. 输入 `cmd` 或 `命令提示符`
3. 点击打开

**方法2：在文件夹中打开**
1. 打开文件夹：`c:\Users\Admins\Documents\记账`
2. 在文件夹空白处**按住 Shift 键**
3. 同时**点击鼠标右键**
4. 选择"在此处打开命令窗口"或"在此处打开 PowerShell"

### 2.2 输入命令（一个字一个字照着输）

在命令窗口中，我会给你一行一行的命令，**复制粘贴**或者**一个字一个字照着输**就行。

#### 命令1：进入项目目录

复制这一行，粘贴到命令窗口，按回车：
```
cd c:\Users\Admins\Documents\记账
```

按回车后，光标应该会变成这样：
```
c:\Users\Admins\Documents\记账>
```

#### 命令2：创建提交

复制这一行，粘贴，按回车：
```
git commit -m Initial
```

**如果报错了，试试这个（带引号）：**
```
git commit -m "Initial"
```

**还是报错？试试这个：**
```
git commit -m ""
```

**按回车后，应该会看到一些英文文字，最后显示：**
```
1 file changed, 1 insertion(+)
create mode 100644 xxxxxx
```

#### 命令3：关联仓库

**这一步要用你刚才复制的地址！**

比如你的仓库地址是：
```
https://github.com/zhangsan/SimpleAccounting.git
```

那就输入：
```
git remote add origin https://github.com/zhangsan/SimpleAccounting.git
```

**把 zhangsan 替换成你的用户名！**

按回车。

如果成功的话，不会有任何提示。

#### 命令4：设置分支

复制这一行，按回车：
```
git branch -M main
```

#### 命令5：推送代码（这是最后一步！）

复制这一行，按回车：
```
git push -u origin main
```

**按回车后，可能会要求输入用户名和密码！**

---

### 🔑 重要：如何输入密码（Token）

当提示输入密码时，**不要输入你登录 GitHub 的密码！**

你需要先创建一个"Token"（密钥）：

#### 如何创建 Token：

1. 打开新标签页，访问：
   ```
   https://github.com/settings/tokens
   ```

2. 点击绿色的按钮：
   ```
   Generate new token → Generate new token (classic)
   ```

3. 填写信息：
   ```
   Note（备注）: Simple Accounting App

   Expiration（过期时间）: 选择 90 days 或 No expiration

   勾选权限：repo（把下面的 repo 全部勾选）
   ```

4. 滚动到页面最底部，点击：
   ```
   Generate token
   ```

5. **复制显示的 token！**
   ```
   类似这样：ghp_xxxxxxxxxxxxxxxxxxxxxx
   ```

**注意：这个 token 只显示一次，一定要复制保存！**

#### 回到命令窗口

1. 当提示 Username 时，输入你的 GitHub 用户名
2. 当提示 Password 时，**粘贴刚才复制的 token**

粘贴密码时，屏幕上可能什么都不显示，这是正常的！直接粘贴就行。

#### 等待上传

按回车后，你会看到很多文字在滚动，类似：

```
Enumerating objects: 150, done.
Counting objects: 100% (150/150), done.
Compressing objects: 100% (80/80), done.
Writing objects: 100% (150/150), done.
...
To https://github.com/你的用户名/SimpleAccounting.git
 * [new branch]      main -> main
```

**看到这些文字，说明上传成功了！** ✅

---

## ⏳ 第三步：等待自动生成APP（5-10分钟）

### 3.1 打开你的 GitHub 仓库

在浏览器中访问：
```
https://github.com/你的用户名/SimpleAccounting
```

### 3.2 点击 Actions

在仓库页面顶部，你会看到几个标签：
```
Code  Issues  Pull requests  Actions  Projects  Wiki
```

点击 **Actions** 这个标签！

### 3.3 查看构建状态

你会看到类似这样的内容：

```
Build Android APK
🟡 In progress（黄色，正在构建）
```

或者：
```
Build Android APK
✅ Build Android APK - Success（绿色，已完成）
```

**刷新页面：**
- 按 F5 键
- 或者点击浏览器上的刷新按钮

**等待 5-10 分钟，直到看到：**
```
Build Android APK
✅ Success
```

所有步骤都变成绿色的对勾 ✅

---

## 📦 第四步：下载APP安装包（1分钟）

### 4.1 点击构建任务

在 Actions 页面，点击最新的绿色 ✅ 任务

### 4.2 滚动到最底部

一直往下滚动，直到看到：

```
Artifacts

📦 app-debug        下载    2.5 MB    2分钟前
📦 app-release      下载    1.8 MB    2分钟前
```

### 4.3 点击下载

点击 **app-debug** 后面的"下载"链接

文件会下载到你的下载文件夹：
```
C:\Users\Admins\Downloads\
```

---

## 📱 第五步：安装到手机（1-2分钟）

### 方法1：最简单的方法

1. **把APK文件传到手机**

   **方式A：用微信**
   - 打开电脑版微信
   - 发送APK文件到"文件传输助手"
   - 在手机上接收并保存

   **方式B：用QQ**
   - 发送APK文件到"我的电脑"
   - 在手机上接收

   **方式C：用百度网盘**
   - 上传APK到百度网盘
   - 在手机上下载

2. **在手机上安装**

   - 找到下载的APK文件（通常在"下载"或"文件管理"里）
   - 点击APK文件
   - 如果提示"允许安装未知应用"，点击"设置"→允许
   - 点击"安装"

3. **打开应用**

   安装完成后，点击"打开"
   或者在桌面上找到"简单记账"图标，点击打开

---

## ✅ 完成！

恭喜你！你的记账APP已经安装到手机上了！

### 🎯 使用APP

打开后，你可以：
- 点击右下角的 **+** 号添加账目
- 点击已有的账目进行编辑或删除
- 点击底部的"统计"查看数据分析
- 点击"我的"导出数据

## ❓ 遇到问题怎么办？

### 问题1：命令行不知道怎么打开？

**解决：**
1. 按键盘 Win 键（田字格）
2. 输入"cmd"
3. 回车
4. 复制命令粘贴进去

### 问题2：git commit 命令一直报错？

**解决：**
试试这几个命令，一个一个试：
```
git commit -m Initial
git commit -m "Initial"
git commit -m ""
```

### 问题3：提示输入密码，但是输入GitHub登录密码不行？

**解决：**
- 不要用GitHub登录密码
- 去创建一个Token（密钥）
- 用Token作为密码
- Token创建教程在上面有详细说明

### 问题4：找不到APK下载链接？

**解决：**
1. 确保构建都完成了（所有步骤都是绿色✅）
2. 刷新页面（F5）
3. 滚动到最底部
4. 看是否有"Artifacts"部分
