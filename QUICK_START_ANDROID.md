# 快速入门指南 - Android版

欢迎使用简单记账Android应用！本指南将帮助您快速上手。

## 🚀 5分钟快速开始

### 步骤1：安装Android Studio

1. 下载并安装 [Android Studio](https://developer.android.com/studio)
2. 安装完成后启动，按照向导完成初始配置
3. 在 SDK Manager 中安装 Android SDK (API 24+)

### 步骤2：打开项目

1. 启动 Android Studio
2. 选择 `File` → `Open`
3. 选择 `记账` 文件夹
4. 等待 Gradle 同步完成（首次可能需要几分钟）

### 步骤3：运行应用

1. 连接 Android 手机并开启 USB 调试，或启动模拟器
2. 点击工具栏的绿色三角形按钮（Run）
3. 应用将安装到设备并自动启动

## 📱 应用功能介绍

### 首页 - 账目

**查看记录**
- 顶部蓝色卡片显示本月支出和收入
- 账目列表显示所有记录，按日期倒序
- 可以筛选全部/支出/收入

**添加记录**
- 点击右下角蓝色的"+"按钮
- 进入记账页面

**编辑记录**
- 点击任意账目进入编辑页面
- 可以修改所有字段或删除记录

### 添加/编辑页面

**选择类型**
- 顶部可以切换"支出"或"收入"

**输入金额**
- 大字号输入框，清晰易读
- 自动限制小数点后两位

**选择类别**
- 8种支出类别（餐饮、交通、购物等）
- 5种收入类别（工资、奖金、投资等）
- 使用Emoji图标，直观易懂

**添加备注**
- 可选的备注字段
- 用于记录额外信息

**保存或删除**
- 点击"保存"按钮保存记录
- 编辑模式下可点击删除图标删除

### 统计页面

**时间选择**
- 点击顶部切换本周/本月/本年
- 自动重新计算统计数据

**查看总览**
- 蓝色卡片显示总支出、总收入、结余
- 结余金额显示颜色（绿色正数、红色负数）

**分类统计**
- 详细列出各类别的金额和占比
- 可视化进度条显示占比
- 按金额从高到低排序

### 我的页面

**统计信息**
- 记账笔数：总共有多少条记录
- 记账天数：记账的不同日期数
- 使用类别：使用了多少种类别

**数据管理**
- 导出数据：将数据复制到剪贴板（CSV格式）
- 清空数据：删除所有记录（需要确认）

**其他功能**
- 关于我们：查看应用信息
- 使用提示：操作建议

## 💡 使用技巧

### 快速记账

1. 打开应用，点击"+"按钮
2. 选择支出或收入
3. 输入金额
4. 点击类别（默认选中第一个）
5. 可选填写备注
6. 点击保存

### 查找记录

- 使用筛选功能快速定位支出或收入
- 按日期倒序排列，最近的在最上面
- 点击记录查看详情

### 数据备份

建议定期导出数据：

1. 进入"我的"页面
2. 点击"导出数据"
3. 数据会复制到剪贴板
4. 粘贴到文本文件或Excel保存

### 统计分析

- 定期查看统计页面了解消费情况
- 切换不同时间周期对比
- 关注占比高的支出类别

## ⚙️ 自定义配置

### 修改类别

如果需要修改类别配置：

1. 打开项目
2. 找到 `app/src/main/java/com/simpleaccounting/app/data/Category.kt`
3. 修改 `expenseCategories` 或 `incomeCategories` 列表
4. 重新构建应用

### 修改主题颜色

如果要修改应用的主题颜色：

1. 在 `ui/` 文件夹中找到各个 Screen 文件
2. 查找 `Color(0xFF4A90E2)` 代码
3. 替换为喜欢的颜色代码
4. 重新构建应用

## 🔧 常见问题

### Q: 如何备份数据？
A: 进入"我的"页面，点击"导出数据"，数据会复制到剪贴板，然后粘贴保存。

### Q: 数据会丢失吗？
A: 数据保存在本地数据库中，不会自动丢失。但卸载应用后数据会被删除，建议定期备份。

### Q: 可以在其他设备上使用吗？
A: 当前版本不支持跨设备同步，每台设备的数据是独立的。

### Q: 如何删除所有数据？
A: 进入"我的"页面，点击"清空数据"，确认后将删除所有记录。

### Q: 应用可以离线使用吗？
A: 完全可以！所有功能都不需要网络连接。

### Q: 支持导出为Excel吗？
A: 导出的是CSV格式，可以直接用Excel打开。
