# 🚀 一键上传 - 超简单使用说明

## 📖 这个脚本做什么？

这个脚本会帮你：
1. ✅ 自动创建 Git 提交
2. ✅ 自动配置 GitHub 仓库
3. ✅ 自动上传代码到 GitHub

**你只需要：双击运行 → 输入用户名 → 粘贴密码 → 等待**

---

## 🎯 使用步骤（3步完成）

### 第1步：双击运行脚本

**两种方式任选其一：**

**方式A：双击这个文件**
```
一键上传.bat
```

**方式B：双击这个文件**
```
一键上传.py
```

（需要安装了Python）

---

### 第2步：按提示操作

脚本运行后，会依次问你：

#### 问题1：请输入你的 GitHub 用户名

**输入你的GitHub用户名，按回车**

比如：
```
zhangsan
```

然后按回车

---

#### 问题2：准备好了吗？

**直接按回车继续**

---

#### 问题3：是否打开仓库网页？

**直接按回车就行**

（会自动打开浏览器）

---

### 第3步：输入密码（Token）

**注意：不是输入GitHub登录密码！**

#### 步骤3.1：创建 Token（密钥）

1. **新开一个浏览器标签页**

2. **访问这个网址：**
   ```
   https://github.com/settings/tokens
   ```

3. **点击按钮：**
   ```
   Generate new token → Generate new token (classic)
   ```

4. **勾选权限：**
   - 找到 **repo**
   - 把下面的所有选项都勾选上

5. **点击底部的按钮：**
   ```
   Generate token
   ```

6. **复制显示的 token**
   ```
   会显示类似这样的一串字符：
   ghp_xxxxxxxxxxxxxxxxxxxxxx

   这个token只显示一次，一定要复制！
   ```

---

#### 步骤3.2：回到脚本

脚本会提示你输入用户名和密码：

1. **Username**
   - 输入你的 GitHub 用户名
   - 按回车

2. **Password**
   - **粘贴刚才复制的 Token**
   - 按回车

**注意：**
- 输入密码时，屏幕上可能什么都不显示
- 这是正常的！
- 直接粘贴就行

---

## ⏳ 等待上传完成

上传过程会显示很多英文文字，类似：

```
Enumerating objects: 150, done.
Counting objects: 100% (150/150), done.
Writing objects: 100% (150/150), done.
...
To https://github.com/你的用户名/SimpleAccounting.git
 * [new branch]      main -> main
```

**看到这些文字，说明上传成功了！** ✅

---

## ✅ 上传成功后

### 自动打开仓库网页

脚本会自动打开你的 GitHub 仓库网页

### 等待自动构建（5-10分钟）

1. 在仓库页面，点击顶部的 **Actions** 标签
2. 等待 5-10 分钟
3. 看到变成绿色的 **✓** 就完成了

### 下载 APK 安装包

1. 点击绿色的构建任务
2. 滚动到页面最底部
3. 找到 **Artifacts** 部分
4. 点击 **app-debug** 下载
5. 传到手机安装

---

## ❓ 常见问题

### 问题1：双击文件没反应？

**解决：**
- 右键点击文件
- 选择"打开方式"
- 选择"记事本"或Python

### 问题2：提示"命令不存在"？

**解决：**
- 确保安装了 Git
- 下载地址：https://git-scm.com/downloads

### 问题3：密码输入错误怎么办？

**解决：**
- 重新创建 Token
- 再次运行脚本
- 重新输入密码

### 问题4：上传失败？

**解决：**
- 检查 GitHub 用户名是否正确
- 检查 Token 是否复制完整
- 再次运行脚本
