# 快速入门指南

欢迎使用简单记账小程序！本指南将帮助您快速上手使用。

## 🎯 第一步：准备图标（必须）

⚠️ **重要**：项目运行前必须先准备图标文件！

在 `images/` 文件夹中放入以下6个PNG图标（81x81像素）：
- tab-home.png（首页，灰色）
- tab-home-active.png（首页，蓝色）
- tab-stats.png（统计，灰色）
- tab-stats-active.png（统计，蓝色）
- tab-mine.png（我的，灰色）
- tab-mine-active.png（我的，蓝色）

**快速获取图标**：访问 [iconfont.cn](https://www.iconfont.cn/) 搜索"账本"、"图表"、"用户"，下载PNG格式即可。

## 📱 第二步：打开项目

1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 微信扫码登录
3. 点击"导入项目"
4. 选择 `记账` 文件夹
5. 填写项目名称，选择AppID（可使用测试号）
6. 点击"导入"
7. 点击"编译"按钮

## 💡 第三步：开始使用

### 添加第一笔账目

1. 在首页点击右下角的 **+** 按钮
2. 切换类型（支出/收入）
3. 输入金额（例如：50）
4. 选择类别（例如：餐饮 🍔）
5. 添加备注（例如：午餐）
6. 选择日期
7. 点击"保存"

### 查看账目列表

- **首页**显示所有账目，按日期倒序排列
- 顶部显示本月收支总览
- 点击"全部/支出/收入"筛选记录
- 点击任意账目可查看详情或编辑

### 查看统计

1. 点击底部"统计"标签
2. 切换时间周期：本周/本月/本年
3. 查看收支总览和结余
4. 查看饼图和分类统计

### 数据管理

1. 点击底部"我的"标签
2. 查看记账统计（笔数、天数、类别）
3. 点击"导出数据"备份到剪贴板
4. 点击"清空数据"删除所有记录（需确认）

- 列表：详细的分类统计

### 我的页
- 用户卡片：显示应用信息
- 统计网格：记账笔数、天数、类别
- 功能列表：导出、清空、关于

## 🛠️ 常用操作

### 编辑账目
1. 在首页点击任意账目
2. 修改需要更改的内容
3. 点击"保存"

### 删除账目
1. 在首页点击任意账目
2. 点击"删除"按钮
3. 确认删除

### 搜索账目
- 当前版本暂不支持搜索功能
- 可以使用筛选功能快速定位

### 导出数据
1. 进入"我的"页面
2. 点击"导出数据"
3. 数据会复制到剪贴板
4. 粘贴到文本文件或Excel中保存

## 💾 数据安全

### 数据存储
- 所有数据保存在本地设备
- 不会上传到服务器
- 隐私安全，无需担心

### 数据备份
建议定期导出数据：
1. 每周或每月导出一次
2. 保存到云端或本地备份
3. 更换设备时可以导入使用

### 数据恢复
- 当前版本不支持直接导入
- 可以手动在代码中修改数据
- 建议保持良好的备份习惯

## ❓ 常见问题

**Q: 为什么图标不显示？**
A: 检查 `images/` 目录中是否有6个PNG图标文件，文件名是否正确。

**Q: 数据会丢失吗？**
A: 数据保存在本地，不会丢失。但更换设备后数据不会同步，建议定期导出备份。

**Q: 如何删除所有数据？**
A: 进入"我的"页面，点击"清空数据"，确认即可删除所有记录。

**Q: 支持多人共享吗？**
A: 当前版本为单用户使用，不支持多人共享账本。

**Q: 可以导出到Excel吗？**
A: 导出的CSV格式可以用Excel打开，选择"数据 > 从文本导入"即可。
