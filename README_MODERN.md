## 🚀 快速开始

### 环境要求
- Python 3.6 或更高版本
- Tkinter（通常Python自带）
- Windows/macOS/Linux

### 安装步骤

#### 1. 检查Python版本
```bash
python --version
```
确保版本是 3.6 或更高。

#### 2. 启动应用

**Windows用户（推荐）：**
```bash
start_modern.bat
```
双击运行 `start_modern.bat` 文件

**命令行方式：**
```bash
python modern_app.py
```

**macOS/Linux用户：**
```bash
python3 modern_app.py
```

## 📖 使用指南

### 添加记账

1. 点击底部的 **"➕ 记账"** 大按钮
2. 选择类型：**支出** 或 **收入**
3. 输入金额
4. 选择类别（会自动显示对应类别）
5. 可选填写备注
6. 选择日期（默认今天）
7. 点击 **"💾 保存"**

### 查看记录

- 主界面显示所有账目列表
- 点击 **"全部/支出/收入"** 按钮筛选
- 点击任意记录可以编辑或删除

### 统计信息

- 顶部卡片显示本月：
  - 💸 **支出总额**
  - 💰 **收入总额**
  - 📊 **结余金额**

结余会根据正负值显示不同颜色：
- 正数：绿色
- 负数：红色
- 零：灰色

## 🎨 自定义样式

### 修改主题色

编辑 `ModernAccountingApp.__init__()` 中的 `self.colors`：

```python
self.colors = {
    'primary': '#667EEA',      # 主色
    'primary_dark': '#764BA2',  # 主色深色
    'expense': '#FF5252',       # 支出色
    'income': '#00C853',        # 收入色
    # ... 其他颜色
}
```

### 修改类别

编辑 `Category` 类：

```python
EXPENSE_CATEGORIES = [
    {'id': 1, 'name': '餐饮', 'icon': '🍜', 'color': '#FF6B6B'},
    # 添加或修改类别
]
```

## 🔧 故障排除

### 问题：Tkinter不可用
**解决方案：**
- Windows：重新安装Python时勾选"tcl/tk and IDLE"
- Linux：`sudo apt-get install python3-tk`
- macOS：通常已包含

### 问题：界面显示异常
**解决方案：**
- 确保Python版本是3.6+
- 检查系统是否支持中文
- 尝试更新Python到最新版本

### 问题：Emoji显示不正常
**解决方案：**
- Windows 10/11：通常正常显示
- 更新系统字体
- 使用Windows 10或更高版本
