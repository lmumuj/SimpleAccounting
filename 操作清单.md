## 🚀 立即开始

### 第一步：创建 GitHub 仓库（2分钟）

1. 访问：https://github.com/new
2. 仓库名：SimpleAccounting
3. **不要勾选** README、.gitignore、License
4. 点击 Create repository
5. 复制仓库地址：`https://github.com/你的用户名/SimpleAccounting.git`

---

### 第二步：上传代码（5分钟）

**方式A：运行脚本（推荐）**
```
双击运行：upload_to_github.py
输入你的 GitHub 用户名
按提示操作
```

**方式B：手动命令**
```bash
cd c:\Users\Admins\Documents\记账
git init
git add .
git commit -m "Initial commit: Simple Accounting App"
git remote add origin https://github.com/你的用户名/SimpleAccounting.git
git branch -M main
git push -u origin main
```

**注意：首次推送需要 Personal Access Token**
- 创建：https://github.com/settings/tokens
- 勾选 repo 权限
- 复制 token 作为密码

---

### 第三步：等待自动构建（5-10分钟）

1. 访问仓库：https://github.com/你的用户名/SimpleAccounting
2. 点击 Actions 标签
3. 等待所有步骤变成 ✅ 绿色对勾

---

### 第四步：下载 APK（1分钟）

1. 点击最新的绿色构建任务
2. 滚动到底部找到 Artifacts
3. 点击下载：
   - app-debug（测试版）
   - app-release（未签名）

---

### 第五步：安装到手机（1分钟）

1. 将 APK 传输到手机
2. 点击 APK 文件
3. 允许安装未知应用
4. 安装并打开
