# 🎯 超简单！3个步骤打包APP（给小白看）

## 📖 快速版教程

如果你觉得上面的教程太长太复杂，看这个就够了！

---

## 步骤1：在网页上创建仓库（2分钟）

### 做什么：
在GitHub网站上创建一个"文件夹"来存放代码

### 怎么做：

1. 打开网址：https://github.com/new
2. 仓库名输入：`SimpleAccounting`
3. 点击绿色的 `Create repository` 按钮
4. **重要**：不要勾选任何选项（README、.gitignore、License都不要勾选）
5. 创建成功后，复制页面上的地址（类似：`https://github.com/你的用户名/SimpleAccounting.git`）

### 完成标志：
- 看到一个新的页面，显示了你的仓库地址
- 地址复制到了记事本里

---

## 步骤2：上传代码（5分钟）

### 做什么：
把电脑上的代码文件传到刚才创建的仓库里

### 怎么做：

**第一步：打开命令窗口**

- 按键盘上的 **Win 键**（田字格图标）
- 输入 `cmd` 然后按回车
- 会弹出一个黑色的窗口

**第二步：复制粘贴命令（一个一个来）**

命令1（进入项目目录）：
```
cd c:\Users\Admins\Documents\记账
```
复制上面这行，粘贴到黑窗口，按回车

命令2（创建提交）：
```
git commit -m Initial
```
复制粘贴，按回车

**如果报错，试试这个：**
```
git commit -m "Initial"
```

命令3（关联仓库 - 用你刚才复制的地址）：
```
git remote add origin https://github.com/你的用户名/SimpleAccounting.git
```
**把 `你的用户名` 改成你真实的GitHub用户名！**

命令4：
```
git branch -M main
```

命令5（上传代码）：
```
git push -u origin main
```

**按回车后会要求输入用户名和密码！**

**密码不是GitHub登录密码！需要先创建一个Token！**

### 如何创建Token（密码）：

1. 新开一个浏览器标签页
2. 访问：https://github.com/settings/tokens
3. 点击 `Generate new token` → `Generate new token (classic)`
4. 勾选 `repo`（全部勾选）
5. 点击 `Generate token`
6. **复制显示的token**（只显示一次，一定要复制！）

### 继续输入密码：

- Username：输入你的GitHub用户名
- Password：粘贴刚才复制的token

**等待几分钟，看到很多英文文字滚动，最后显示 "main -> main" 就成功了！**

### 完成标志：
- 命令窗口显示上传成功
- 刷新你的GitHub仓库页面，能看到很多文件

---

## 步骤3：下载APP安装包（5-10分钟）

### 做什么：
等待GitHub自动帮你把代码变成APP安装包

### 怎么做：

**第一步：打开Actions页面**

1. 访问你的GitHub仓库
2. 点击顶部的 `Actions` 标签
3. 等待5-10分钟
4. 看到变成绿色的 ✅ 就完成了

**第二步：下载APK**

1. 点击绿色的构建任务
2. 滚动到页面最底部
3. 找到 `Artifacts` 部分
4. 点击 `app-debug` 后面的下载链接
5. 文件会下载到下载文件夹

**第三步：安装到手机**

1. 把下载的APK文件传到手机（微信/QQ/网盘都可以）
2. 在手机上点击APK文件
3. 点击"安装"
4. 安装完成后，点击"打开"

### 完成标志：
- 手机上多了"简单记账"这个APP
- 可以打开使用了

恭喜你，你的记账APP已经安装到手机上了！

## 🆘 救命！遇到问题了？

### 问题1：命令窗口不知道怎么打开？

**解决：**
- 按键盘 Win 键（田字格）
- 输入"cmd"
- 回车

### 问题2：git commit 命令报错？

**解决：**
试试这两个命令：
```
git commit -m Initial
git commit -m "Initial"
```

### 问题3：密码输入GitHub登录密码不行？

**解决：**
- 不要用登录密码
- 创建一个Token（密钥）
- 用Token作为密码
- Token创建：https://github.com/settings/tokens
- 勾选 repo 权限

### 问题4：找不到下载链接？

**解决：**
- 确保构建都完成了（所有步骤都是绿色✅）
- 刷新页面（F5）
- 滚动到最底部
- 找到 "Artifacts" 部分
