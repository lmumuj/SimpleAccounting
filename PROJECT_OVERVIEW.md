# 简单记账小程序 - 项目概览

## 项目简介

**项目名称**：简单记账
**项目类型**：微信小程序
**适用平台**：安卓、iOS
**设计风格**：简约明朗、清新舒适
**版本**：1.0.0

## 核心功能

### 1. 首页 - 账目管理
- 展示本月支出和收入总览
- 支持筛选全部/支出/收入记录
- 账目列表按日期倒序排列
- 点击账目可查看详情或编辑
- 快速添加账目按钮

### 2. 添加/编辑账目
- 支出/收入类型切换
- 大字号金额输入，清晰易读
- 图标化的类别选择
- 备注输入
- 日期选择
- 编辑模式下支持删除记录

### 3. 数据统计
- 时间周期切换：本周/本月/本年
- 收支总览和结余计算
- 饼图展示各类别占比
- 详细的分类统计列表
- 可视化进度条

### 4. 个人中心
- 记账统计（笔数、天数、类别）
- 数据导出（CSV格式）
- 清空数据功能
- 使用提示
- 关于信息

## 技术架构

### 技术栈
- **前端框架**：微信小程序原生框架
- **开发语言**：JavaScript
- **样式语言**：WXSS
- **数据存储**：wx.setStorageSync/wx.getStorageSync
- **图表绘制**：Canvas API

### 页面结构

```
pages/
├── index/        # 首页
├── add/          # 添加/编辑
├── statistics/   # 统计
└── mine/         # 个人中心
```

### 核心数据结构

#### 账目记录（accountBooks）
```javascript
{
  id: Number,              // 唯一标识
  type: String,            // 'expense' | 'income'
  amount: Number,          // 金额
  categoryId: Number,      // 类别ID
  remark: String,          // 备注
  date: String,            // 日期
  createTime: String       // 创建时间
}
```

#### 类别配置（categories）
```javascript
{
  expense: [
    { id: 1, name: '餐饮', icon: '🍔' },
    { id: 2, name: '交通', icon: '🚗' },
    // ... 共8个支出类别
  ],
  income: [
    { id: 101, name: '工资', icon: '💰' },
    { id: 102, name: '奖金', icon: '🎁' },
    // ... 共5个收入类别
  ]
}
```

## 常见问题

### Q: 数据会丢失吗？
A: 数据存储在本地，不会丢失。但更换设备后数据不会同步，建议定期导出备份。

### Q: 如何自定义类别？
A: 可以修改 `app.js` 中的 `initData()` 方法，添加或修改类别配置。

### Q: 支持多人使用吗？
A: 当前版本为单用户使用，不支持多人共享账本。

### Q: 如何修改主题颜色？
A: 修改 `app.wxss` 和各页面 `.wxss` 文件中的颜色代码。
