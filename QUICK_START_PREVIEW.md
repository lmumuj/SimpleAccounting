# 快速开始 - Python预览版

## 🚀 3步启动

### 第1步：检查Python

打开命令行，输入：

```bash
python --version
```

确保显示 Python 3.6 或更高版本。

**如果没有安装Python**：
- 访问 https://www.python.org/downloads/
- 下载并安装Python 3.8或更高版本
- **重要**：安装时勾选 "Add Python to PATH"

### 第2步：运行应用

**方法一：双击运行（Windows）**

直接双击 `run_preview.bat` 文件

**方法二：命令行运行**

```bash
python preview_app.py
```

**方法三：右键运行**

右键点击 `preview_app.py`，选择 "Open with Python"

### 第3步：开始使用

应用启动后，您可以：

1. **添加账目**：点击右下角"+ 记账"按钮
2. **查看记录**：查看列表中的所有账目
3. **统计分析**：点击"📊 统计"查看数据
4. **管理数据**：点击"👤 我的"导出或清空数据

## 💡 第一次使用

### 添加第一笔账目

1. 点击右下角的"+ 记账"按钮
2. 选择类型：支出
3. 输入金额：例如 50
4. 选择类别：餐饮 🍔
5. 备注可选：午餐
6. 日期：默认今天
7. 点击"保存"

### 查看记录

- 主界面会显示您刚添加的记录
- 顶部卡片会更新本月支出
- 点击记录可以查看详情或编辑

### 查看统计

1. 点击底部"📊 统计"按钮
2. 选择"本月"
3. 查看支出和收入统计

## 📊 数据说明

### 自动保存

- 所有数据自动保存到 `transactions.json` 文件
- 文件和应用在同一目录
- 关闭应用不会丢失数据

### 备份数据

建议定期备份：

1. 复制 `transactions.json` 文件
2. 保存到其他位置或云盘

### 导出数据

1. 点击"👤 我的"按钮
2. 点击"📤 导出数据"
3. 数据会复制到剪贴板
4. 粘贴到文本文件或Excel保存

## ⚠️ 常见问题

### Q: Python不是内部或外部命令？

A: Python未正确安装或未添加到PATH

**解决方法：**
1. 重新安装Python，确保勾选 "Add Python to PATH"
2. 或直接双击 `preview_app.py`（如果关联了Python）

### Q: Tkinter not found？

A: Tkinter未安装

**解决方法：**
- **Windows**：重新安装Python，勾选 "tcl/tk and IDLE"
- **Linux**：`sudo apt-get install python3-tk`
- **macOS**：`brew install python-tk`

### Q: 数据丢失了？

A: 检查是否有 `transactions.json` 文件

**解决方法：**
- 文件可能被移动或删除
- 建议定期备份数据文件
- 重新添加账目即可

### Q: 界面显示不全？

A: 屏幕分辨率或缩放问题

**解决方法：**
- 调整屏幕缩放为100%
- 使用更高分辨率的屏幕
- 或修改代码中的窗口大小
