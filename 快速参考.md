# 🚀 GitHub Actions 快速参考

## 📋 一页纸教程

```
1. 下载 Git
   https://git-scm.com/downloads

2. 注册 GitHub
   https://github.com

3. 创建仓库
   New repository → SimpleAccounting → Create

4. 上传代码
   cd c:\Users\Admins\Documents\记账
   python upload_to_github.py
   或双击：upload_to_github.bat

5. 等待构建
   GitHub 仓库 → Actions → 等待 5-10 分钟

6. 下载 APK
   Actions → 最新任务 → Artifacts → 下载

7. 安装到手机
   传输到手机 → 点击 APK → 安装
```

---

## 🔑 关键链接

- Git 下载：https://git-scm.com/downloads
- GitHub 注册：https://github.com
- 创建仓库：https://github.com/new
- 创建 Token：https://github.com/settings/tokens

## ✅ 成功标志

构建成功后会看到：
```

✅ Upload Debug APK
✅ Build Release APK
✅ Upload Release APK

Artifacts:
  app-debug (2.5 MB)
  app-release (1.8 MB)
```

点击文件名即可下载！

## 📱 安装命令

```bash
# ADB 安装
adb install app-debug.apk

# 覆盖安装
adb install -r app-debug.apk

# 卸载
adb uninstall com.simpleaccounting.app
```
